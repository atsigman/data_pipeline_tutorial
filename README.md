# Data Pipeline Tutorial

This repo is intended for pedagogical purposes only. 

To download the relevant assets, install requirements, and execute the data pipeline, follow the instructions in the `music_data_pipeline_tutorial` Colab notebook, included in the parent directory. 

To use locally: 
  1) Clone the repo
  2) `cd data_pipeline_tutorial`
  3) Create a venv
  4) `pip install -e .`
  5) Use the Colab notebook as a guide for pipeline operations and `AudioDataset` functionality.
